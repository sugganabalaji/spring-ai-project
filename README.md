# 🌟 Spring AI Multi‑Model Project
## 🚀 Overview
This project demonstrates how to integrate `Spring Boot` with `Spring AI` to orchestrate multiple AI capabilities:
- 🔮 **OpenAI & Ollama** for LLM‑powered responses
- 📊 **Embeddings + Vector DB** for semantic search and retrieval
- 🎙️ **Audio & Speech Models** for transcription and text‑to‑speech
- 💬 **Chat Models** for conversational AI APIs
- 🎨 **AI Image Generators** for creative visual outputs
It’s designed as a cloud‑ready, project showcasing modern backend practices and multi‑modal AI integration.
